这是一个条件判断语句。使用以下形式：
　{if, notif} 条件表达式 ({&&, &} 条件表达式) 执行语句
if 在条件成立时执行后续语句，notif 在条件不成立时执行后续语句。
如果条件不成立，该行将被跳过。
&& 和 & 的含义相同，用于进一步限定条件。
条件表达式可以使用数字或 fchk 函数。
　　　＜条件表达式＞
　　　数字 {>, <, =, >=, <=, ==, !=, <>} 数字
　　　或者，
　　　fchk 字符串

　　　<, >, =, >=, <= 的含义显而易见。
　　　== 与 = 相同，表示「相等」；!= 与 <> 相同，表示「不相等」。
　　　fchk 如果曾经加载过指定字符串的图像，则返回真。

　　　　示例）
　　　　if %0=1 quakex 2　（基本形式）
　　　　if %0=1 && %2=5 end　（两个条件同时成立时执行）
　　　　if %0=2 mov %1,1:skip 2 （使用 : 连接多行命令）
　　　　notif %0=1 quakex 2　（条件不成立时执行）
